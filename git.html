Neste espaço consta anotações sobre o treinamento de git/github do bootcamp
O git é uma ferramenta de versionamento de software, ele é muito utilizado, e serve como facilitador, quando utilizamos a programação em pares, isso por que o versionamento é controlado através de commits, e caso mais de uma alteração no código seja realizada simultaneamente, o git exibirá uma crítica e pedirá ao usuário administrador para selecionar qual versão será publicada.

já o Github é uma plataforma que hospeda o sistema de controle de versionamento git. é o local onde os repositórios ficam armazenados e são disponibilizados para colaboração.
